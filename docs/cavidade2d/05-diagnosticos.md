# Diagnósticos de Estabilidade

**Reynolds muito alto** → use resolução maior.
Frequência de relaxação  

$$  
\omega = \frac{1}{3\,\nu_l + 0.5}  
$$


$$
\nu_l = \frac{\nu}{c_s^2\,\Delta t}
$$
  Deve obedecer $0 < \omega < 2$, onde $c_s = 1/\sqrt{3}$.
